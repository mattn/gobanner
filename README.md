# gobanner

banner written in golang

## Usage

```
Usage of gobanner:
  -f string
    	path to TTF file
  -s int
    	font size (default 15)
  -w int
    	screen width (default 79)
```

## Installation

```
go get github.com/mattn/gobanner
```
